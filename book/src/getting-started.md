# Getting Started

This guide will help you get started with AITK.

```admonish success
By the end of this guide, you'll have AITK integrated into your project!
```

## Installation

Add AITK to your `Cargo.toml`:

```toml
[dependencies]
aitk = "0.1.0"
```

```admonish tip
Check [crates.io](https://crates.io/crates/aitk) for the latest version.
```

## Basic Usage

```admonish example
Here's a simple example to get you started:

\`\`\`rust
use aitk::*;

fn main() {
    // Your code here
}
\`\`\`
```
