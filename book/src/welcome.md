# Welcome to AITK

AITK is a Rust crate providing cross-platform and framework-agnostic abstractions to work with AI/LLMs.

```admonish info
This book provides comprehensive documentation for AITK, including guides, examples, and API references.
```

## Features

```admonish tip
AITK works seamlessly on desktop, mobile, and web (WASM). It can be used in CLI apps, GUI apps, servers, and more!
```

- Cross-platform support (desktop, mobile, web)
- Framework-agnostic design
- Type-safe abstractions for AI/LLM interactions
- Multiple provider support
- State management utilities
